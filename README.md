gym-azul, an openAI gym environment for the Azul board game
===========================================================

# Installation

Create a python3 venv and activate it
```
virtualenv -p python3 venv
source venv/bin/activate
```

Install gym-azul
```
git clone git@gitlab.binets.fr:drakaena/gym-azul.git
cd gym-azul
pip install .
```
